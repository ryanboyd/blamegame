# BlameGame: Sentence-Level Attribution Detection

**BlameGame** is a Python package for detecting internal/external (dispositional/situational) attributions in text at the sentence level. Whether you're exploring psychological patterns or analyzing narrative frameworks, BlameGame combines cutting-edge NLP models and regression techniques to detect the reasons that people give for explaining their own (and other people's) behaviors.

---

## 🎯 Features

- **Sentence-Level Attribution**: Detects situational and dispositional attributions within individual sentences.
- **Regression-Powered Analysis**: Applies pre-trained Ridge Regression models to predict attribution probabilities.
- **Seamless Processing**: Handles large datasets with ease and outputs results in CSV format.
- **Vectorized Outputs**: Can output raw sentence vectors for any additional modeling/applications.
